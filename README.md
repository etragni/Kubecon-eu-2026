# API is the New SSH: Forging a Zero-Trust VM Platform on Kubernetes

This repository contains the presentation slides for the session "API is the New SSH: Forging a Zero-Trust VM Platform on Kubernetes" delivered at KubeCon Europe 2026.

### 📋 Session Abstract

By 2026, the landscape of virtualization has shifted toward unified infrastructure where virtualization is a feature, not a place. 

This presentation explores the transition from legacy hypervisors to KubeVirt, treating Virtual Machines as "Legacy Pods" within a single orchestration layer.
The session covers the architectural shift from manual SSH-based management to a Zero-Trust posture where VMs are managed as first-class Kubernetes citizens, sharing the same RBAC, storage (CSI), and networking (CNI) pipelines as containerized microservices.
